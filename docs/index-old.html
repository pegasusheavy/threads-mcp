Quick Installation

1

Install Dependencies

pnpm install threads-mcp
2

Set Environment Variables

export THREADS_ACCESS_TOKEN="your-token"
export THREADS_USER_ID="your-user-id"
3

Configure Claude Desktop

{
  "mcpServers": {
    "threads": {
      "command": "threads-mcp",
      "env": {
        "THREADS_ACCESS_TOKEN": "your-token",
        "THREADS_USER_ID": "your-user-id"
      }
    }
  }
}
